#### Vacform Housing Parts
| Part | Part Number | Qty | Description |
|------|-------------|-----|-------------|
| 1 | Fire rating vacform housing | - | - |

| 8 | M4 Rivnuts | - | - |
| 2 | LDR with housing | - | - |
| 2 | LDR cables | - | - |

##### Assembly Steps
- Install pinch weld rubber
- Install 16mm grommets
- Drill out 16mm grommets with 5mm drill bit
- Install LDRs
- Feed matrix data and power leads thru grommets (use lubrication)
- Install vacfrom on to fire rating base plate
- Install dual solar panel mount connectors
- Wire matrix data cable to terminal rail
- Wire matrix power cables to terminal rail
- Plug in LDR cable

##### Assembly Steps
- Clean down aluminium plate and inside of door skin with IPA
- Cover both surfaces with contact adhesive and all to skin over (5-10mins)
- Align both parts and stick together (use roller to remove any air bubbles)
- Install foam sear around edge
- Install condensation valves
- Install door sticker
- Attach to sign
